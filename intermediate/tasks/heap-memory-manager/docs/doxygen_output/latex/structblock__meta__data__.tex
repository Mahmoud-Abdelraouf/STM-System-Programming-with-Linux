\doxysubsection{Detailed Description}
Structure representing metadata for a memory block. 

The {\ttfamily block\+\_\+meta\+\_\+data\+\_\+t} structure represents metadata for a memory block. It includes information such as whether the block is free or allocated, its size, pointers to the previous and next blocks (if applicable), and the offset within the memory region.
