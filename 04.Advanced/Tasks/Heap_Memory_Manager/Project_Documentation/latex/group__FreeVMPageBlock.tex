{\footnotesize\ttfamily void xfree (\begin{DoxyParamCaption}\item[{void \texorpdfstring{$\ast$}{*}}]{app\+\_\+data }\end{DoxyParamCaption})}



Frees memory allocated by the memory manager. 

This function is used to free memory that was previously allocated by the memory manager. It takes a pointer to the memory to be freed as its argument. The pointer is adjusted to point to the block metadata, and then it is passed to the memory manager\textquotesingle{}s free blocks function.
